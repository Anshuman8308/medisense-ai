About MediSense AI

MediSense AI v2 is a production-ready healthcare intelligence platform combining classical ML with a clean, modular web application to provide AI-assisted medical guidance.

ML Pipeline
Random Forest · sklearn Pipeline (Scaler + RF) · 5-fold CV · Precision, Recall, F1 · Confusion matrix · Feature importance
Architecture
Frontend: HTML/CSS/JS · Backend: Flask (modular blueprints) · Services layer · Standard JSON API · Environment config
⚕️ Medical Disclaimer
MediSense AI is built for educational and demonstration purposes only. All AI-generated predictions are NOT medical diagnoses and must NOT replace professional medical advice, diagnosis, or treatment. Always consult a qualified healthcare professional.
